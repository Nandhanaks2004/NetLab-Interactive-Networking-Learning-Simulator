# NetLab: Interactive Networking Learning Simulator

NetLab is an interactive web-based learning platform designed to teach networking fundamentals through visual simulations, real-world analogies, and hands-on exercises.

The application focuses on helping beginners understand how networks work—from basic concepts to advanced topics like protocols, addressing, and subnetting—through interaction rather than static content.

---

## Project Overview

Networking concepts are often difficult for beginners because they are abstract and hard to visualize. NetLab solves this by converting theoretical topics into interactive simulations and step-by-step visual flows.

This project is designed as a teaching tool for students to explore how data travels across networks and how different components interact in real-world scenarios.

---

## Objective

The goal of NetLab is to help learners:

* Understand how devices communicate in a network
* Visualize how data travels from a user to a server
* Learn key networking concepts through interaction
* Build a strong foundation for cybersecurity and ethical hacking

---

## Key Features

### Network Fundamentals

* Introduction to networks using real-world analogies
* Visual representation of device communication

### Network Types and Topologies

* PAN, LAN, MAN, WAN explained interactively
* Star, Bus, Ring, and Mesh topology visualizations

### Network Devices and Data Flow

* Interactive flow showing how data moves through:

  * Switch
  * Router
  * Firewall
* Packet movement animations

### Protocol Flow Simulation

* Step-by-step simulation of:

  * DNS resolution
  * ARP process
  * TCP handshake
  * HTTP request and response
* Interactive explanation of each stage

### OSI and TCP/IP Models

* Clickable layered diagrams
* Mapping between OSI and TCP/IP models
* Layer-wise explanation with examples

### TCP vs UDP

* Visual comparison
* TCP 3-way handshake animation

### IP Addressing

* Explanation of IP vs MAC address
* Public vs Private IP
* Static vs Dynamic IP

### Subnetting Lab (Core Feature)

* Interactive subnetting calculator
* Step-by-step guided subnetting example
* Visual subnet splitting
* Practice mode for students

Includes:

* Host calculation formula: 2^(host bits) - 2
* Subnet calculation logic
* CIDR-based subnet breakdown

### CIDR and Routing

* CIDR notation explanation
* Classful vs Classless comparison
* Static vs Dynamic routing overview

### Network Security Basics

* Introduction to:

  * Firewall
  * Proxy
  * VPN
  * Load Balancer
* Visual representation of secure data flow

### End-to-End Network Simulation

* Complete flow visualization:
  User → DNS → ARP → TCP → HTTP → Server → Response
